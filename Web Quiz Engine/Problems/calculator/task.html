To consolidate your knowledge of REST API, you need to develop a web calculator with three operations: addition, subtraction and multiplication.

To perform these operations, the web calculator must accept GET requests using the following urls:

  • /add
  • /subtract
  • /mult

The requests will contain two arguments a and b in the request parameters. Here is an particular example: /add?a=10&b=20.

As a response, the calculator must return a result value and the 200 OK HTTP code. If there is an unsupported operation specified in the link (e.g. /divide), the service must return the string "Unknown operation".

We've already prepared a template for you. Just add some Spring annotations, their imports and code to perform calculations. Please, do not remove the calculator package!
