Description

Online chat is an application that lets you send and receive messages through the Internet. This project will show and explain the magic behind casually chatting with your online pals: you will work that magic yourself!

The first stage of the project is pretty simple. Imagine that some people are messaging each other in a group chat. Your program should receive raw messages from the standard input, process them, and output only the text type messages ignoring the technical messages like when someone joined/left the chat.

Each line of the input contains the person's name and the action this person performed. If the person performed the sent action, then the rest of the line after the keyword sent is the text of their message. Check out the output example to better understanding!

Objectives

In this stage, your program should satisfy the following conditions:

  1. Every line of the input contains the person's name and the action this person performed (sent, joined, left, connected, and disconnected).
  2. The person's name is a single word; there are no character restrictions for the personâ€™s name.
  3. The sent messages match the pattern "Name sent Message".
  4. The messages in the output match the pattern "Name: Message".
  5. All other types of messages are not shown in the output.
  6. Your program will receive only correct raw messages.
Example

For the following input:

Nick joined
Nick sent Hello everyone!
Nick sent Someone here?
Josh joined
Josh sent Hey Nick!
Nick sent Hey Josh! I thought no one here...
Josh sent Oh, I just joined, so..
Nick sent OK
Nick disconnected
Josh sent Are you here?
Nick connected
Nick sent Yes, just bad connection
Josh sent Alright
Nick left
Josh left

The program should output:

Nick: Hello everyone!
Nick: Someone here?
Josh: Hey Nick!
Nick: Hey Josh! I thought no one was here...
Josh: Oh, I just joined, so..
Nick: OK
Josh: Are you here?
Nick: Yes, just bad connection
Josh: Alright
