Description

At this point, our program can download and display the source code of a web page, but what about the title? It would be very nice if our users could see the page title right away. This is exactly what we are going to implement in this stage, but first, let's refresh something about HTML code.

As you saw in the previous stage, HTML code contains many angle bracket symbols, < and >. These symbols declare tags. So, HTML consists of tags. There are two types of tags:

  • Pairs comprised of a start tag "<tagname>" and an end tag "</tagname>". These tags act like containers, and content is placed between these paired tags. For example, the title of a web page can look something like this: <title>Some Title</title>. Note that these tags are used not only for declaring titles.
  • Empty-element tags, for example, <img src="LINK" />, a tag used for inserting images. Again, this type of tag is not only used for inserting images. The only way to customize these tags are attribute additions.
    Attributes are pairs of an attribute name and an attribute value. For example, you can see the attribute src="LINK" in the img tag. It can also be written as src='LINK'.

Now that you know a bit about HTML tags, you need to obtain the title of a web page and display it in a JLabel. To do this, you need to extract the content between the start tag <title> and the end tag </title>.

Note that in the example below, the word "Title:" and the title itself are in two separate JLabel.

Objectives

In this stage, you should:

  1. Add a JLabel to the window.
  2. Set the text of the JLabel to the title of the web page when the run button is clicked.
  3. Set the names of the following components:
  • JTextField "UrlTextField";
  • JButton "RunButton";
  • JLabel "TitleLabel";
  • JTextArea "HtmlTextArea".
