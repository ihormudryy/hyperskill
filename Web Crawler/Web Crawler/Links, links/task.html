Description

Currently, our window contains only a text area that shows text set at the start of the program. This kind of functionality is not very impressing, so let's take it one step further: add the ability to download the source code of a web page and display it in the text area.

So, how do we download the source code? There are standard Java classes that can help us with that.

Here is a code snippet that gets the source code of a web page: 

final String url = /* Get url from JTextField */;

try (InputStream inputStream = new BufferedInputStream(new URL(url).openStream())) {
String siteText = new String(inputStream.readAllBytes(), StandardCharsets.UTF_8);
}

Note that the code to open InputStream is placed in a try-with-resources construct, so inputStream will be closed automatically.

IMPORTANT: For testing reasons, please do not interact with your computer while tests are running. Also if you're running on a Mac OS, make sure you give Intellij access to control your computer. To do so please follow these steps. Go to System Preferences -> Security & Privacy -> Accessibility -> Intellij IDEA CE.app and check the checkbox to allow your IDE to control your computer.

Objectives

In this stage, you should:

  1. Add a JTextField to the window for entering a URL.
  2. Add a JButton for initiating the download and displaying the source code in the text area.
  3. Set the names of the following components:
  • JTextField "UrlTextField";
  • JButton "RunButton";
  • JTextArea "HtmlTextArea".
