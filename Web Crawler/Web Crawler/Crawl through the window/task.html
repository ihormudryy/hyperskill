Description

Let's start developing the crawler. In this stage, we are going to create a basic window containing a single text area that displays static text. In further stages, we will improve our bot to behave the way we want.

You should add a JTextArea that displays the text "HTML code?".

If you want the full list of components, you can find it on zetcode.com in part I and part II.

Feel free to position the components wherever you like throughout this project.

Objectives

In this stage, you should:

  1. Set the title of the window to "Web Crawler".
  2. Add a JTextArea to the window.
  3. Place the text "HTML code?" into the text area and make it disabled.
  4. Set the name of the component JTextArea "TextArea".
