Description

As you know, web pages contain links to other web pages. In this stage, we are going to collect all the links to other web pages and their respective titles from a given web page and display them in a JTable.

In HTML, links are tags like <a target="_blank" href="LINK">, where LINK is a URL that can be contained in single (') or double (") quotes.

The link itself can be presented in different ways:

  • An absolute link;
  • A relative link;
  • A link without a protocol.

Don't forget that other than links to web pages, there are also links to images, audio files, videos, and so on. Let's handle these cases by updating the page download code.

The URL's openStream() method is shorthand for openConnection().getInputStream(). Receive an object URLConnection by openConnection() and check its getContentType() which can return a null value so be sure to handle such a case. If it starts with "text/html", then use .getInputStream() to pass the connection to the InputStreamReader.

When the run button is clicked, the data should be downloaded. Display this data in the JTable with the first column being the link and the second column the title. Make sure to clear the previous data in the table if any.

The first row in the table should contain the initial page and its title.The remaining rows should contain links found within the page paired with their titles. And you need to make sure that the links you add to your table are formatted in their absolute form.

You should put the JTable in a JScrollPane so that the JTable header is shown.

You can implement the table in Swing using JTable.

Objectives

In this stage, you should:

  1. Add a JTable to the window and put it in a JScrollPane.
  2. Display the data collected in the JTable.
  3. Disable the JTable.
  4. Set the names of the following components:
  • JTextField "UrlTextField";
  • JButton "RunButton";
  • JLabel "TitleLabel";
  • JTable "TitlesTable".
